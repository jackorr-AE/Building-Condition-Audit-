# Reference Files

This folder contains the source documents and assets used to generate the formatted building inspection reports.

## `Example Report.pdf`
Example report containing the desired formatted tables you want to reproduce.

Use it as the formatting reference for:
1. Table layout and ordering
2. Column structure
3. Any consistent headings, sections, and visual styling that should appear in the generated output

## `Fulcrum Export.xlsx`
Contains the data collected from a recent inspection of a building and surrounding assets.

It includes:
1. General rooms
2. External area conditions
3. Individual assets for testing
4. Defects that require remediation

Use this as the data source that the generated report should pull from, mapping the appropriate fields into the tables shown in `Example Report.pdf`.

## Logos (for reporting)
These two logos should be used in any reporting output:

1. `AWB Logo.png`
2. `Asset Engineers Logo.png`

Use the same logo pair consistently anywhere you generate report header/branding sections.
